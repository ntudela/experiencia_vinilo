# Experiencia Vinilo - Página de Sugerencias

Este es un prototipo en React (Next.js) para sugerir discos según lo que busca el cliente.

## ¿Cómo correrlo?

1. Instala dependencias:
   npm install next react react-dom

2. Ejecuta el servidor de desarrollo:
   npm run dev

3. O súbelo directo a vercel.com
