# Vinilo Catalog

Mini‑sitio React/Next.js para explorar tu catálogo de discos.

## Uso

1. Exporta tu hoja de cálculo a **catalogo.json** con las columnas:
   `Nombre Disco`, `Autor`, `Categoria`, `Década`, `Género`, `Año`.
2. Reemplaza el archivo en **public/**.
3. Instala dependencias y arranca:

   ```bash
   pnpm install   # o npm install
   pnpm run dev
   ```

4. Sube este repositorio a GitHub y despliega en Vercel.
